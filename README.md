# üåä Flood Control Project Satisfaction Survey

A full-stack web application designed to gather taxpayer feedback regarding recent flood control projects in the region. This project demonstrates a modern client-server architecture using **FastAPI** for the backend and a responsive **HTML** interface for the frontend.


## ‚öôÔ∏è How to Run

1.  **Clone the Repository**
    ```bash
    git clone [https://github.com/nic340/survey_project.git](https://github.com/nic340/survey_project.git)
    cd survey_project
    ```

2.  **Install Dependencies**
    ```bash
    pip install fastapi uvicorn sqlmodel
    ```

3.  **Start the Server**
    ```bash
    uvicorn main:app --reload
    ```
    *The API will be live at `http://127.0.0.1:8000`*

4.  **Launch the Survey**
    * Open `index.html` in your web browser.
    * Fill out the form and submit.
    * View saved data at `http://127.0.0.1:8000/data`.
